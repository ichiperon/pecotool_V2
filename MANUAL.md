# 📄 PecoTool V2 取扱説明書 (v1.0.0)

PecoTool V2 をご利用いただきありがとうございます！
このツールは、PDFファイルのテキスト情報を効率的に抽出・編集・コピーするための業務効率化ツールです。

---

## 🚀 1. インストール方法

1.  配布フォルダにある `pecotool-v2_1.0.0_x64_en-US.msi` (または最新版のMSIファイル) をダブルクリックします。
2.  画面の指示に従ってインストールを完了させてください。
3.  デスクトップまたはスタートメニューから **PecoTool V2** を起動します。

---

## 🛠 2. 基本的な使い方

### ① PDFの読み込み
- ツールバーの **「ファイルを開く」** ボタンから、対象のPDFファイルを選択します。
- ページが読み込まれ、キャンバスに表示されます。

### ② 領域の選択（OCRカードの作成）
- マウスでPDF上のテキスト範囲をドラッグして囲みます。
- 囲んだ範囲が「カード」として右側のリストに追加されます。

### ③ テキストの編集とコピー
- 右側のカードリストで、抽出されたテキストを自由に編集できます。
- **「コピー」** ボタンを押すと、クリップボードにテキストがコピーされます。
- カードの順番をドラッグ＆ドロップで入れ替えることも可能です。

### ④ 保存と書き出し
- 作業状態を **「保存」** して、後で再開することができます。
- 全カードのテキストをまとめてプレビューしたり、ファイルとして保存することも可能です。

## ⌨️ 3. 便利なショートカットキー

作業をさらにスピードアップさせるための秘密のコマンドよ！

| ショートカット | アクション |
| :--- | :--- |
| `Ctrl + O` | PDFファイルを開く |
| `Ctrl + S` | 現在の状態を保存 |
| `Ctrl + +` | ズームイン |
| `Ctrl + -` | ズームアウト |
| `Ctrl + 0` | 画面のサイズにフィット |
| `Delete / BackSpace` | 選択中のカードを削除 |
| `↑ / ↓` | カードの選択を切り替え |

---

## ❓ 4. 困ったときは

- **「テキストがうまく読み込めない！」**
  - 画像としてスキャンされたPDFの場合、現在はテキスト抽出ができません。今後のアップデートをお楽しみに！
- **「動作が重い…」**
  - 非常にページ数の多いPDFや、高解像度の画像が含まれる場合に重くなることがあります。その場合は、必要なページだけ分割してから読み込んでみてください。
